Primeiro Projeto na Trybe

O projeto "Lições Aprendidas" foi realizado durante o curso de Desenvolvimento de Software na Trybe. A grade curricular do curso conta com mais de 1.500 horas de aulas que abordam a introdução ao desenvolvimento de software, front-end, back-end, ciência da computação, engenharia de software, metodologias ágeis e habilidades comportamentais.

Esta página é resultado de muito aprendizado e consolida mais uma etapa na minha formação como desenvolvedora. Foram apenas três semanas de curso mas são conhecimentos e vivências para toda a vida. Durante este período nossa turma teve muitas interações que foram essenciais para conhecer um pouco mais sobre a Trybe, a sua equipe de profissionais e nossos colegas de sala. Além disso, aprendemos sobre o Unix e o Shell Script. Não foi uma tarefa fácil mas é gratificante superar os desafios de trabalhar com o terminal.

A realização e avaliação deste projeto também não seria possível se não tivessemos aprendido sobre o Git e o Github. No início parece difícil mas para o crescimento de um desenvolvedor estas ferramentas são essenciais. Para finalizar, aprendemos os conceitos básicos de HTML e CSS que nos permitiram chegar nesta etapa do projeto. Tenho muito orgulho e admiração das minhas vitórias e de fazer parte desta escola.
