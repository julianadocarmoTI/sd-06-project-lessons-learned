Mil lições em três semanas: O que eu aprendi na Trybe

O que eu aprendi até agora na Trybe:

  1. Unix&Bash: aprendi a utilizar o terminal de forma correta, sendo muito mais produtiva;
  2. Shell-scripts: essa foi a primeira linguagem que aprendi, muito útil para automatizar tarefas corriqueiras;
  3. O maravilhoso mundo da Internet: aprendi conceitos básicos sobre o funcionamento da internet;
  4. Git&GitHub: aprendi a utilizar de forma correta o Git e o GitHub - como criar repositórios, adicionar arquivos, fazer alterações (adeus versao_8_final_submetido!), fazer code reviews e muito mais!
  5. HTML: aprendi a criar uma página web do zero;
  6. CSS: aprendi a adicionar mais estilo a minha página web;
  7. Aprendi a deixar o meu LinkedIn mais profissional;
  8. Aprendi sobre a importância da inteligência emocional no mercado de trabalho;
  9. Aprendi a aprender...pois é, o desenvolvimento Web é uma novidade para mim! Aprendi como organizar os meus estudos, como fazer buscas eficientes no Google, aprendi a trabalhar remotamente...e continuo aprendendo para me desenvolver cada vez mais nessa área que tanto me encanta!
  10. Aprendi que as lições da Trybe vão muito além da programação e desenvolvimento de Software! Conheci pessoas incríveis sem tê-las conhecido pessoalmente, pratico diariamente exercícios de autoconhecimento que me ajudam a traçar meus caminhos e percebo como o aprendizado que a Trybe nos proporcionam vão muito além do desenvolvimento Web.
