Minha Maratona de Aprendizado

Na nossa maratona de conhecimentos, começamos por adequar nosso sistema operacional para trabalhar com desenvolvimento web, foi solicitado a ciação do dual boot nas nossas respectivas máquinas, sendo já de largada uma ótima oportunidade de conhecer e trabalhar com o linux, agradeço muito meu irmão Vitor, pois meu pen drive de dual boot dava "boot" toda hora. Pra mim foi super oportuno e embora já tivesse conhecimentos prévios sobre o linux, trabalhar com desenvolvimento web foi uma ótima oportunidade, eu trabalhava com máquinas virtuais mas pude ver que realmente o linux nativo tem um rendimento superior.No tocante a shell script eu nao tinha conhecimentos prévios dele, echo e /bin foi uma novidade boa, mais uma ferramenta pra colocar na caixa, sendo que foi muito bom ter esse contato. Estava esquecendo que previamente trabalhamos com comandos no terminal linux, foi bom lembrar dos comando cat, mv, mkdir entre outros para com isso ter mais bons conhecimentos para mais ferramentas ainda na nossa caixinha de skills, nos preparando para o mercado de trabalho como um todo. Trabalhamos com git e github para versionar e hospedar alterações que criamos, sendo possível também importar de outros desenvolvedores outros códigos e nos ajudar mutuamente. Cabe dizer que o git irá nos acompanhar durante todo o curso e sempre tem um comandinho secreto pra aprender não é mesmo Agora estamos trabalhando com o HTML e CSS para começarmos a trilha de desenvolvedores de páginas web, muito feliz com os resultados e ansioso para continuar esta maratona. Já tinha um conhecimento deles, mas é semore bom ver novamente para internalizar nossos conhecimentos, estamos ansiosos para logo mais "tricotar" páginas responsivas para todas as telinhas Claro que não podemos esquecer da Amanda e do RL, não basta ser um bom dev tecnicamente, mas também devemos nos esmerar para trabalhar bem em equipe, sendo que está experiência de utilizar o zoom está sendo fantástica, pois de verdade sinto que todos estamos juntos acredito muito no potencial dos outros colegas e vamos nos destacar no mercado de trabalho.#GoTrybe
