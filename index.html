Iniciando...

A primeira coisa que fizemos foi preparar nosso computador para o trabalho. Isso incluiu, primeiramente instalar o Ubuntu 20.04 LTS. Com o sistema operacional instalado, também instalamos outros programas, para que o fluxo de trabalho funcionasse melhor. Foram eles: Google Chrome, como navegador. VSCode, como interface de programação. Zoom, como ferramenta de videoconferências, por onde todas as aulas acontecem. Slack, um ambiente de interação e compartilhamento, onde todos os avisos são postados, incluindo a agenda do dia, links para o forms (explicado abaixo) e arquivos.
